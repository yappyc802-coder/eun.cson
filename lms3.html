Section 1: Types of CSS

CSS can be applies in three ways: Inline, Internal, and External. Inline styles are attributes that override Inernal and External. Internal styles are writtenin the <head> section and overrides External styles stored in a seperate .css file. These help make wep pages easier to design and maintain.

Section 2: Display Property and Element Types

The display property controls how elements appear on a webpage. Block elements like <div> and <p> stay on a new line, while inline elements like <span> and <a> stay in the same line. These tags help organize layout and visibility in your website.

Section 3: Div and Span Tags

The <div> and <span> tags are containers used for grouping and styling content. The <div> tag is for larger sections while <span> tag is for smaller text parts. They work best when combined with CSS for better layout control.
