STYLE SHEETS: pt.1

Section 1: Structure and Purpose of CSS

Lesson 2 explains the structure and key parts of a CSS rule, which include the selector, property, value, and declaration. CSS uses symbols like curly braces { } , colons : , and semicolons ; to define styles. A selector targets the HTML element to style, while properties describe what to change (like color or font), and values specify how it should look. Together, they form declarations inside a declaration block. A style sheet is simply a set of instructions that tell the browser how to display HTML elements, separating content (HTML) from design (CSS) for a cleaner and more organized webpage.

Section 2: Types of Style Sheets

  • External CSS is stored in a separate .css file with the use of the <link> tag and linked to multiple HTML pages, making design changes faster and consistent across a website.
  • Embedded CSS is placed inside the <head> section of an HTML file using <style>, applying styles to that single document. This overrides External.
  • Inline CSS is written directly within an HTML tag using the style attribute, affecting only one element. This overrides Externak and Embedded.

The lesson also introduces CSS comments (/* comment */)> which help explain code for easier editing and understanding. Overall, this lesson emphasizes how CSS simplifies web design by allowing control, reusability, and organization of styles.
