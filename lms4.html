CSS RULES, CLASSES, AND SELECTORS

Section 1: CSS Rules and Properties

CSS rule is made up of a selecctor and a list of declarations that define how elements should look. The selector targets the HTML element, and the declarations contain properties and values that control style aspects like fonts, colors, text, layout and lists. CSS can be written inline, internal or external, depending on how it's applied onto a wepbage.

Section 2: Classes and Element + Class Selectors

A class is a user-defined selector to style specific elements using the class attribute. Classes are written with a dot . before their names, like .highlight, and can be reused acrss different HTML tags. An element + class selector (e.g., p.note) targets only a specific element with a particular class, giving more precise control over styling.

Section 3: Grouping Selectors

Grouping Selectors lets you apply the same style to multiple elements in one line by seperating them with commas (e.g., h1, p {color:blue;}). This helps to make your codes shorter, cleaner and easier to manage. It also improves consistency when styling multiple HTML elements that share similar designs.
